# AI Agent Workflow Skill

这是一个中英双语 AI skill 仓库，目标是：帮助用户围绕“AI agent workflow skill opportunities”构建可复用 AI skill 工作流，并基于最新资料给出步骤、边界和验证清单。

## 安装 / Install

将本仓库克隆到你的本地 skill 目录，或复制 `SKILL.md` 到目标工具的 skill 目录。

```powershell
git clone https://github.com/TheAatroxking1/ai-agent-workflow-skill.git
```

## 快速开始 / Quick Start

1. 打开你的 AI coding 工具。
2. 启用 `ai_agent_workflow_skill` skill。
3. 提供你的目标、输入材料和希望输出的格式。
4. 让工具按 `SKILL.md` 中的流程完成任务。
